eRent Mobile

eRent Mobile is a iOS application for online rent management and payment solutions for Ann Arbor landlords and property management firms. It provides the platform where residents, property managers and technicians can communicate on property maintenance.

The residents can submit work orders if they need help. Then the manager assign the work order to a technician. Technicians can view the work orders that are assigned to them, and go to the specific unit and fix the fixture accordingly. Also, users can submit comments and pictures to provide more details about the fixture. This platform makes residents get problems solved conveniently, helps technicians work efficiently and allows managers to monitor and record their property easily.

It's based on a MySQL database and a RESTful web service in PHP MVC mode. Frameworks that were used include JQuery, PhoneGap, JQueryMobile and CodeIgniter. It uses AJAX to exchange data with the server through the RESTful web service. Also, Git was used as the version control tool.
